# RS AI - Advanced AI Operating System Assistant

RS AI is a high-performance, completely offline, and free AI assistant for Windows.

## Features
- **Local AI**: Integration with Ollama for offline LLM capabilities.
- **Voice I/O**: Multi-language support with auto-detection.
- **High-Res Image Generation**: Professional 8K/16K ultra-high-resolution images with Realistic, Anime, Supernatural, and Movie styles.
- **Supernatural Video & Movie Generation**: Create cinematic videos and full movies with character consistency.
- **Character Engine**: Create and manage unique characters with specific looks, ages, genders, and unique AI voices.
- **Dynamic AI Audio**: Voice generation tailored to character attributes (age, gender, type).
- **Web Finder**: Precisely designed web search tool to find and scrape information from the internet.
- **OS Control**: Control your Windows environment.
- **Offline & Free**: No API keys required.

## Development
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python src/main.py`
